# 🏆 Sport Management System

## 📌 Project Overview

The **Sport Management System** is a web-based application developed using PHP, HTML, CSS, and MySQL. It helps users manage sports activities, bookings, and basic administration tasks.

---

## 🚀 Features

* User registration & login
* Sports selection (Cricket, Football, etc.)
* Booking system
* Admin panel for management
* Payment integration (Razorpay - test mode)

---

## 🛠️ Technologies Used

* Frontend: HTML, CSS
* Backend: PHP
* Database: MySQL
* Server: XAMPP / Localhost

---

## 💻 How to Run the Project

### 1. Install Requirements

* Install XAMPP (or any PHP server)
* Start **Apache** and **MySQL**

### 2. Setup Project

* Copy project folder to:

  ```
  C:\xampp\htdocs\
  ```

### 3. Setup Database

* Open phpMyAdmin
* Create a new database (e.g., `sports_db`)
* Import the provided `.sql` file

### 4. Configure Database Connection

* Open `db_connect.php`
* Update:

  ```php
  $username = "root";
  $password = "";
  $dbname = "sports_db";
  ```

### 5. Run Project

Open browser and go to:

```
http://localhost/sport-management-system/
```
